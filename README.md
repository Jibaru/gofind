# Concurrent finder tool

## Usage:

```
gofind -d <directory> -s <search string> -m <max number of coincidences>
```

## Flags

```
  -d string
        directory to search (default "./")
  -h    see help
  -l string
        log file (default "coincidences.json")
  -m int
        max number of coincidences (default 100)
  -s string
        search string
```
